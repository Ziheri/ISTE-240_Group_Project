Requirements & Setup

Requirements:

You need a computer with an operating system

  • Window
  • MacOS
  • Linux

Setup:

For windows

Windows Subsystem for Linux (WSL) to make a unix like environment

For the full unix environment you would need a vm (virtual machine)

For MacOS

Using terminal to install the code line tools xcode-select --install and /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com)" **this is installing homebrew**

UTM (VM to run a separate unix operating system..)

For Linux

Install the NixOS and get the GNOME or Plasma ISO

Install nix package manager (on existing linux)

File System Hierarchy

Contains both standard files and directories in a tree-like structure starting from the root directory (the current drive you are on)

You are able to access it either through GNOME System monitor or the CLI (command line interface)

Security & Permissions

The file system has to read, write and execute to different users with different privileges.

Administration privileges (sudo permissions)
