History of Unix

What is Unix?

UNIX is a collection of operating systems that serve as a precursor to several modern operating systems. Several prolific operating systems such as Linux, Mac OS X, Android etc. use its design principles as a basis.

Beginnings

UNIX started development in 1969 by Ken Thompson and Dennis Ritchie at AT&T Bell Labs. The first version was finished in 1971 and was able to be used on an early minicomputer, the PDP-11/20 Its primary function was processing the text of patent documents. It included several features that are standard in operating systems even in the modern day such as a hierarchical file system, which allowed users to arrange files in individual directories that could store other directories. It did not, however, include a graphical user interface (GUI). It was an early open source program, as users were encouraged to modify and iterate upon the source code.

In 1973 it was rewritten in C and made portable, meaning it could be used on multiple different platforms. In 1974, a paper about it was released in Communications of the ACM that made interest surge rapidly. AT&T wasn't legally allowed to sell anything besides telecommunications technologies at this time, so the source code was licensed for a small fee. Because of this, some of the first support groups for UNIX usage appeared. In 1983, UNIX System V was the first commercially available version of UNIX. These two landmarks in its development made it so that UNIX was made widely available.

UNIX-based systems today

UNIX serves as a basis for how software is used and presented in the modern day, which gives it a large amount of historical significance. Variations of UNIX are used today primarily for servers and workstations but UNIX-based systems are commonly used today for daily computing tasks.
